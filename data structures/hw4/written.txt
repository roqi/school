5 -

a) For a min heap, every parent is smaller than the children. We also know that
the maximum is the max of maxs of every path down the tree.
Now, we know that all the maxs of every path down the tree is the last element
(the leaf). Since the ultimate max is just one of them, it must be a leaf too.

b) We know that the last leaf is the last element of the array (n). Its parent
is the element in the middle (index floor(n/2) ). So all elements between
floor(n/2 +1) and n have no children, so they are leafs.
Hence, the number of leafs is n - floor(n/2) = ceil(n/2)

c) The tree maximum is the max of the maxs of every path down the tree.
Each max of a path down the tree is the last element in that path (the leaf).
So to find the tree maximum, all of the maxs of the paths must be compared,
so all of the leafs must be examined and compared.
